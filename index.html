HOW TO USE THE OVERLAY

1

Select the Case you are about to open in the top-right menu.

2

Press Ctrl + Shift + P to activate the "Ready" mode (the dot will turn green). Or just click on the Ready status.

3

Click the "Open Case" button in CS2. The overlay will detect it and start the animation.

4

Use "Quick Open" if you want to skip the scroll animation.

!

If the overlay is not reading skin names properly, make sure to Calibrate it with the button in the top-right corner.

Use Fullscreen windowed, if you can't see overlay in game.
